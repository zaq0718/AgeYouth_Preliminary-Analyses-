#  AgeYouth Meta-Analysis Project

> _Summary:_  
> *“This project performs a comprehensive meta-analysis on age and youth well-being data, including data cleaning, transformation to long format, moderator analysis, and visualization via R Shiny.”*

---

## 📁 Repository Structure
---

## 🧩 Main Components

### **1. Data**
- `data/raw/` — Original raw datasets used in the analysis.  
- `data/processed/` — Cleaned, merged, and standardized files used for modeling.

### **2. Notes and Variable Descriptions**
- Located in `meta_analysis_notes/`.  
- Contains detailed notes on:
  - Variable meanings and transformations.  
  - Step-by-step documentation of meta-analysis workflow.  
  - Changes made to raw data (added/removed IDs, corrections, etc.).

### **3. Output**
- Contains processed outputs and intermediate data products generated by `.Rmd` scripts.  

### **4. R Markdown Analysis Files**

| File | Purpose |
|------|----------|
| **AgeYouth_MetaAnalysis.Rmd** | Main experimental and commented meta-analysis with all results. |
| **AgeYouth_MetaAnalysis_additionalCHE.Rmd** | Moderator analysis and mean-level change models based on Rooney’s code and suggestions. |
| **country_wiseID_youth.Rmd** | Generates country-wise ID-level summary output for the complete dataset. |
| **long_format_seq.Rmd** | Converts raw data into **long sequential format** required for modeling. |
| **mean_diff.Rmd** | Computes mean differences and visualizes their patterns; produces output dataset. |
| **new_longformat_nonseq.Rmd** | Converts only **non-sequential interval** data into long format. |
| **qualitycoding_ageyouth.Rmd** | Cleans and prepares the quality-coding dataset for the Shiny app. |
| **raw_score.Rmd** | Used for experimental runs and testing within the meta-analysis pipeline. |

### **5. Shiny Application**
- Folder: `meta_analysis_shiny/`  
  - Contains all **old and new Shiny app versions**, supporting processed files, and final cleaned outputs for interactive visualization of meta-analysis results.
