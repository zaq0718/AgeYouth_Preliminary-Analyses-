# AgeYouth: Preliminary Analyses 
## Project Overview
This project examines the development of well-being during adolescence.
The data, code, and results available in this repository are part of the **preliminary analyses** and reflect the project status as of **October 2026**. They should not be considered the final data, code, or results associated with the publication.

## Accessing the Preliminary Materials
To access the preliminary data, code, and results:
1. You are now under the branch **`main`**.
1. Click the branch selector near the top-left of the repository page.
2. Select the branch named **`dev-clean`**.
3. The relevant data, code, and results will then be available within that branch.

Please note that the materials may be revised before publication.
